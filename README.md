#  سامانه BNPL - پنل فروشندگان

# نحوه اجرا
1. XAMPP را اجرا کنید
2. فایل `database.sql` را در phpMyAdmin import کنید
3. پوشه پروژه را در `htdocs` قرار دهید
4. در مرورگر آدرس `http://localhost/bnpl-panel` را باز کنید
